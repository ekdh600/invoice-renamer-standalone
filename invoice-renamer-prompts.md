# Prompts (현재 사용 중)

이 문서는 `invoice-renamer`가 **현재 코드에서 그대로 사용 중인 프롬프트**를 기록합니다.

> 위치: `src/invoice_renamer/cli.py`

## 1) 로컬 VLM OCR 프롬프트 (이미지 → 텍스트)

LM Studio `POST /v1/chat/completions`에 전달합니다.

```text
이 이미지는 견적서/문서야. 문서 안의 모든 텍스트를 그대로 추출해줘. 손글씨(한글) 포함해서 읽을 수 있는 글자는 전부 적어줘. 설명 없이 추출한 텍스트만 출력해.
```

추가 동작:

- 이미지가 큰 경우 실패를 줄이기 위해 **긴 변 1536px 리사이즈 + JPEG 품질 85**로 전송

## 2) 로컬 LLM 추출 프롬프트 (텍스트 → JSON: 상호/날짜)

LM Studio `POST /api/v1/chat`에 전달합니다.

```text
아래 견적서 텍스트에서 발행업체 상호명과 날짜를 추출해.
- 반드시 JSON만 출력: {"company_name":"...","date":"YYYYMMDD 또는 Unknown"}
- 상호는 '상호/업체명' 라벨의 회사명 우선
- 날짜는 YYYYMMDD 형식, 없으면 Unknown
[견적서 텍스트]
{text[:6000]}
[텍스트 끝]
```

## 3) 정규식 기반 1차 추출(프롬프트 아님)

프롬프트 호출 전에 다음 라벨 주변에서 정규식으로 1차 후보를 추출합니다.

- `상호`
- `업체명`
- `업체` (업체명 제외)

날짜는 다음 라벨 우선:

- `작성일`, `견적일`, `발행일`, `날짜`, `년월일`
